CSS的三大特性

1.层叠性

相同选择器给设置相同的样式，此时一个样式就会覆盖（层叠）另一个冲突的样式。层叠性主要解决样式冲突的问题。

原则：就近原则，样式不冲突的不会覆盖。

2.继承性

CSS中的继承：字标签会继承父标签的某些样式，如文本颜色和字号

例如：

<head>
<style>
   div{
       color: pink;
  }
</style>
</head>
<body>
<div>
<p>前端前端</p>
</div>
</body>

子元素一般继承的属性（text-,font-,line-以及color属性）

行高的继承表示的特殊形式（不跟单位）:

<style> body{ font:12px/1.5 ‘Microsoft YaHei’ } div{ font-size:14px;/*1.5即当前元素文字大小的1.5*/ } </style>

优点：让<body>中的子元素可以根据自己文字的大小自动调节行高

3.优先级

*选择器相同，则执行层属性（就近原则）。

*选择器不同，则根据选择器权重执行。

继承或*（0,0,0,0,）<元素选择器（0,0,0,1）<类选择器、伪类选择器)(0,0,1,0)<id选择器(0,1,0,0)<行内样式style=“”(1,0,0,0)<!important 重要的（无穷大）

\<style>

       .test {
           color: red;
      }
       div {
           color: bule!important;
      }
#demo{
  color: green;
}
</style>

注意：1.权重是4组数组组成，但是不会有进制。

           2.等级判断从左向右，如果某位数值相同，则判断下一 位数值。

           3.继承的权重是0，如果该元素没有直接选中，不管父元 素权重多高，子元素得到的权重都是0。（先看有没 有被直接选出）（a浏览器默认的样式为蓝色有下划 线，若继承父元素，无效。）

权重叠加：如果是复合选择器，则会有权重叠加，需要我们去计 算权重。

例：.nav li{} 0,0,1,0+0,0,01=0,0,1,1

盒子模型

（盒子模型、浮动、定位）

网页布局过程：

*准备好相关的网页元素，网页元素基本都是盒子

*利用CSS设置好盒子的样式，然后放置到相应的位置！！！！

*往盒子里面装内容

盒子模型：把HTML中的布局元素看成是盒子，也就是一个橙装内 容的容器。

CSS盒子模型本质是盒子，封装周围的HTML元素，包括:边框(border)，外边距(margin)，内边距(padding)，和实际内容(content)

<1>border

组成属性：边框宽度（粗细）、边框样式、边框颜色

（border-width）（border-style）(border-color)

*样式：solid(实线)、dashed(虚)、dotted(点)、等等。

*复合写法：border：width style color；(没有先后顺序)

*边框的分写：border-top(上边框)：width style color;

也可利用样式的层叠性(上框红色，其他三条蓝色).

        \<style>

               div{
              width: 100px;
              height: 100px;
                   border: 10px solid blue;
                   border-top: 10px soild red;
                }
</style>

对表格进行border设置

    \<style>

           table{
               width: 500px;
               height: 100px;
          }
           table,
           td，
           th{
               border: 10px solid pink;
               border-collapse:collapase;(合并相邻 单元格，不然相邻单元格之间的线比较粗)
          }
</style>

边框会改变盒子的大小

            \<style>

                   div{
                       width: 180px;
                       height: 180px（盒子的宽高）
                       background: bule;
                       border: 10px solid red;
                  }
</style>此盒子的宽，高均为180，而不是200。

<2>内边距padding

padding-bottom(方位词): 10px;

padding的复合属性：

padding: 5px；(上下左右)

padding: 5px 10px；(上下为5，左右为10)

padding: 5px 10px 20px (上为5，左右为10，下为20)

padding：5px 10px 20px 30px（上右下左）

**padding影响盒子大小

如果盒子已经有了宽高，此时在指定内边框，会撑大盒子。

应用：新浪导航栏练习

padding不会撑开盒子的情况：盒子没有书写width/height

<3>内容content

<4>外边距margin(方位)

控制盒子与盒子之间的距离。

应用：可以让块级元素实现水平居中的效果

（1）盒子必须指定了宽度

（2）盒子的左右的外边距都设置为auto

margin: 0 auto;

补充：

***让行内元素或者行内块元素水平居中：

直接在加text-align： center；

***外边距合并（块级元素）

***对于两个嵌套关系（父子关系）的元素，父元素有上外边 距同时子元素也有上外边距，此时父元素会塌陷较大的外边距值

解决方案：

（1）可以为父元素定义上边框（border-color：transpa）

（2）可以为父元素定义内边距

（3）可以违法元素添加overflow：hidden

补充：清除网页自带的内外边距

<style> *{ padding: 0; margin: 0; } </style>

注意：行内元素为了照顾兼容性，尽管只设置左右内外边距，不要设置上下内外边距。但是转换为块级和行内块元素就可以了。

圆角边框

border-radius：length；（圆与边框的交集形成的圆角效果）

规律：数值越大，弧度越来越大（radius为盒子宽度的一半时为圆）

<1>参数值可为数值或者百分比

<2>盒子为正方形，想设置成一个圆，将宽或高修改为50%或 者一半即可

<3>盒子为长方形，设置为高度的一半就可以做圆角边框

<4>该属性是简写属性，可以跟四个值，左上、右上、右下、 左下 （跟两个：左上与右下、左下与右上）

<5>也可设置一个border-top-left-radius:20px;

盒子阴影

box-shadow: h-shadow v-shadow blur spread color inset;

(前两个必需有，且分别为水平，垂直，都允许为负值)

(后几个可选，分别是模糊距离，阴影的尺寸，阴影的颜色<rgba（0,0,0,.3）>，将外部阴影<outset默认>改为内部阴影）

原先没有影子，当我们经过盒子的时候就添加阴影效果。

div（元素标签）：hover {

box-shadow： 10px 10px 10px -4px black；

}

文字阴影：

text-shadow：h-shadow v-shadow （必需）

blur，color（可选）
