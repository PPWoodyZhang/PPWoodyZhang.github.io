11月28日CSS基础笔记

CSS中的属性名和HTML中的属性名有所不同

css简介：美化页面（html美化页面很麻烦，css操作方便，也是一门标记语言）

css语法规范：css规则由两部分组成：选择器，一条或者多条声明

一般写在<head>与</head>之间，用<style></style>,模板是 选择器{样式}

选择器是用于指定Css样式的html标签

属性与属性值以“键值对”的形式出现color：red；《分号结束，非常重要》

CSS的代码风格（良好的习惯）：

1.样式格式书写

<1>紧凑格式 h3 {color: red;font-size: 12px;}

<2>(推荐)展开格式 h3 {

color: pink;

font-size: 12px;

}

2.样式大小写(采用小写)

3.空格规范

<1>属性值前面，冒号后面，保留一个空格

<2>选择器（标签）和大括号中间保留空格

CSS的基础选择器：

作用：根据不同的需求把不同的标签选出来（选择标签）

1.基础选择器（由单个选择器组成）

<1>标签选择器（html标签名称作为选择器,把一类标签全部选择出来，所有的<p>）

《优点》：快速为页面中同类型的标签统一设置样式

《缺点》:不能具体到一类中的某一个

<style>

标签名{

属性名1：属性值1；

属性名2：属性值2；

………

}

</style>

<2>类选择器

《优点》：一类标签中单独的选一个或几个

<style>

.类名（一般不用html标签名来给类起名，长名称或词组可以用中横线来为选择器命名，不要使用纯数字、中文等命名。尽量使用英文字母来表示）

{

属性1：属性值1；

………….

}

</style>

                \<div class='red（类名）'>变红色\</div>

类选择器的口诀：样式点定义，结构类（class）调用 ， 一个或多个，开发最常用

类选择器的多类名：一个标签有多个类名

            \<div class="red font20">亚瑟\</div>

要求：1.在class属性中写多个类名

2.多个类名中间必须用空格隔开

使用场景：将相同的样式放到一个类中

这些标签都可以调用这个公共类，然后再调用自己独有的类，节省CSS代码

<3>id选择器（标有特定id的html元素指定特定的样式）

html元素以id属性来设置id选择器，CSS中的id选择器以“#”来定义。

#id名{

属性1：属性值；

…….

}

口诀：样式通过#定义，结构id调用，只能调用一次，别人切勿调用

姓名是类选择器，身份证号是id选择器（使用次数）

类选择器在修改样式中用的最多，id选择器一般用于页面唯一性的元素上，经常和Javasript搭配使用

<4>通配选择器(不需要去调用，自动将所有进行改变)

使用*(html,span,body,div,li)来定义它表示选取页面中所有的元素（标签）

<style>

*{

属性1：属性值；

}

</style>

2.复合选择器

CSS字体属性

CSS Fonts(字体)属性用于定义字体系列，大小，粗细，和文字样式（如斜体）

1.font-family属性定义文本的字体系列

p{font-family: ‘微软雅黑（Microsoft YaHei)‘,Arial,Helvetica,’Times New Roman’}；【找对人，做对事】

各个字体之间必须使用英文状态下的逗号隔开，对于由多个单词构成的字体，要用引号隔开（单双均可）

尽量使用电脑默认的字体，保证都能显示

写多个字体，兼容性更好

\<style>
    body\{font-family:'Microsoft YaHei',tahoma,arial;\}
\</style>

2.font-size属性定义字体大小

p{

font-size=20px;

}

px(像素)大小是我们网页的最常用单位，谷歌默认显示16px

可以给body指定整个页面文字的大小，最好设置，不要用浏览器默认

标题标签比较特殊，需要单独指定文字大小

3.font-weight属性定义字体粗细

<head>

    \<style>

          .bold{
              /* font-weight:bold;*/
              font-weight:700;(更常用)
          }
</style>

</head>

<body>

​

我嫩爹

​

</body>

特殊情况：标题本身会加粗，不想让其加粗，可以用normal或者400；

4.font-style属性设置文本的风格

<style>

p{

font-style:normal;

}

</style>

normal:默认值，浏览器显示的标准的字体样式

italic：浏览器会显示斜体的字体样式

最常用的是将斜体换成正常的

em（i）{

font-style：normal；

}

_我嫩爹_>

字体的复合属性（可以把以上文字样式综合来写，节约代码）

<style> div{ font-style=italic; font-weight=700; font-size=16px; font-family=’Microsoft yahei’ } </style>

写成了(简写的形式）div{font：font-style font-weight font-size/line/height font-family;

}

div{font:italic 700 16px ‘Microsoft yahei’}

不需要的属性可省略，但必须保留font-size和font-family，否则font属性将不起作用

文本属性（文本的颜色，对齐文本，装饰文本，文本缩进，行间距等）

1.文本颜色

<style> div{ color;pink; 或者 color:#ff0000（实际用吸色管吸取颜色）; 或者 color：rgb(255,0,0)(红色)

</style>

2.文本对齐

text-align属性用于设置元素内文本内容的水平对齐方式（左对齐，右对齐，水平居中）

<style> ​ div{

text-align：center(right,left(默认值));

}

</style>

3.文本装饰

text-decoration属性规定添加到文本的修饰。可以给文本添加下划线，删除线，上划线等。

添加和去掉线

<head>

<style>
  div{
      text-decoration: underline;
  }
  a{
  text-decoration:none;
  }
</style>
</head>
<body>
<a href=”#”>我嫩爹</a>
</body>

none(默认没有装饰)，underline（常用），overline（几乎不用），line-through（删除线，不常用）

4.文本缩进

text-ingent属性用来指定文本的第一行的缩进，通常是讲段落的首行缩进。

div{

text-indent:10px(也可为负值，负值往左，正值往右);

}

了解单位 em是一个相对单位，就是当前元素（font-size）1个文字的大小，如果当前元素没有设置大小，则会按照父元素（css继承）的一个文字大小。

5.行间距（上间距，文本间距，下间距）真正的通过改变上下间距来改变。

p{

line-height：26px；（改变上下间距）

}

若文字是16px，line-height=26px，则是对半分。“检查”点击的框即是全部。

CSS的引入方式(前面只讲了CSS可以写在<style>内)

根据CSS书写位置（引入方式的不同）CSS样式表可以分为三大类：

1.内部样式表（嵌入式）

将CSS写到HTML内部，是将所有的CSS代码抽出来单独放在一个<style>标签中

\<style>

      div{
          color=red；
          font：font-style font-weight font-size/line/height font-family;
      }
</style>

理论上<style></style>可以放在HTML文件的任意位置，一般放在<head></head>

优点：代码结构清晰，但是没有实现结构与样式完全分离。

2.行内样式表（内联样式表）（行内式）

是在元素标签内部的style属性中设定CSS样式，适合于修改简单样式

<div style=”color: red; font-size: 12px;”>我嫩爹</div>

style其实就是标签的属性，要在双引号中间，可以控制当前行的样式。（不提倡大量使用）

3.外部样式表（链接式）

实际开发都是外部样式表，适用于样式比较多的情况。

优点：多个页面可以使用

核心：样式单独写到CSS文件中，之后把CSS文件引入到HTML页面中使用。

两步走：

《1》建立一个只有样式的.css文件

div{

color：red；

}

《2》使用<link>标签（写进head部分）将.css文件引入.html文件中。

<link rel=”stylesheet” href=”css文件路径”>
