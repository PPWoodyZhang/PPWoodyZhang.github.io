11月29日，学习ing...........,跟上PINK老师的步伐

1.快速格式化代码（emmet语法）

        设置：保存页面是就自动格式化代码

      （1）文件—–【首选项】——-【设置】

      （2）搜索emmet.include

      （3）在settings.json下的【用户】中添加以下语句：

        “editor.formationOnType”:true,

        “editor.formationOnSave”:true

        只需要设置一次即可，以后都可以自动保存格式化代码

     （vscode也可使用快捷键：shift+alt+f）

2.复合选择器（基础选择器组合而成）

（1）后代选择器（包含选择器）

        可以选择父元素中的子元素

        语法：元素1 元素2 （元素3..）{声明样式}（多个标签选择器拼接，元素1是元素2的父元素，结果改变元素2的样式）

        ol li {声明样式}

        注意：元素2可以是元素1的子元素，也可为孙辈元素，只要是后代即可。

        这里的元素可以是任意的选择器。

                    \<style>

                           ul li{
                               color: red;
                          }
                           .ab li a{
                               color: pink;
                          }
</style>

（2）子选择器(只能选择作为某元素的最近一级子元素，即亲儿子元素)

        元素1>元素2{样式声明}

（3）并集选择器（可以选择多组标签，同时为他们定义相同的样式，通常用于集体声明）

        元素1，元素2………{声明样式}

        注意：<1>元素1与元素2之间用英文逗号（可理解为“和”）隔开,最后一个不用逗号。

                   <2>约定的规范，并集选择器一般竖着写

（4）伪类选择器（用于向某些选择器添加特殊效果，例如给链接添加特殊效果，或选择第一个，第n个元素）

        <1>链接伪类选择器

        a:link /选择所有未被访问的链接/

        a:visited /选择所有已访问的链接/

        a:hover /选择鼠标指针位于其上的链接/

        a:active /选择活动链接（鼠标按下未弹起的链接）/

        注意：**为确保生效，按照“LVHA”的顺序声明

        **a链接在浏览器中有默认的样式，所以开发中要单独指定样式。

实际开发常用样式：

            \<style>

                   a{
                       color: red;
                       text-decoratin: none;
                  }
                   a：hover{
                       color: bule;  
                       text-decoretion: underline;
                  }
</style>

<2>focus伪类选择器（用于选取获得焦点的表单元素）

        焦点就是光标，一般情况<input>类表单元素才能获取，因此这个选择器也主要针对于表单元素来说.

            \<style>

                  input:focus{
                      background-color:yellow;
                  }
</style>

3.元素的显示模式（元素（标签）以什么方式进行显示，比如<div>自己占一行，或者一行可以放多个<span>)

<1>块元素（<div>，<h1>,<h6>,<p>,<ul>,）

        （1）独占一行

        （2）高度，宽度，外边距以及内边距都可以控制

        （3）宽度默认是容器（父级宽度）的100%

        （4）是一个容器及盒子，里面可以放行内或者块级元素

        注意：文字类的元素不能使用块级元素（<p><h1><h6>）

<2>行内元素/内联元素(<a>,<strong>,<b>,<em>,<i>,<del>,<s>,<ins>,<u>,<span>)

        (1)相邻的行内元素在一行上，一行可以显示多个

        (2)高，宽直接设置是无效的

        (3)默认宽度就是它本身内容的宽度

        (4)行内的元素只能容纳文本或者其他行内元素

        注意：链接里面不能再放链接,特殊情况链接<a>里面可以放块级元素，但是给<a>转换一下块级元素更安全

<3>行内块元素：同时具有块级元素和行内元素的特点<img/><input/><td>

        (1)相邻行内元素可以放在一行，但是之间会有空白间隙（行内）

        (2)默认宽度就是它本身内容的宽度（行内）

        (3)高度，行高，外边距以及内边距都可以控制（块级）

4.CSS的元素显示模式转换（一个模式的元素需要另外一种模式的特性）

       <1>行内元素转化为块级元素（display:block;）

        <2>块级元素转化为行内元素（display:inline;）

        <3>其他元素转化为行内块元素（display:inline-block;）

5.snipaste工具的使用（截图工具）

        (1)f1可以截图同时测量大小，设置箭头，书写字符

        (2)f3在桌面置顶显示

        (3)点击图片，alt+c可以取色，（按下shift可以切花取色模式）

      （4）按下esc取消图片的显示

小技巧：单行文字垂直居中（即垂直方向上文字边距相同）（让文字的行高等于盒子的高度）line-height：**px；

6.CSS的背景

       通过背景属性来设置背景颜色，背景图片，背景平铺，背景图片的位置，背景图像的固定等。

           <1>背景颜色background-color:transparent(默认)/color

           <2>背景图片background-image(常见于logo或者一些小的装饰的小图片或者是超大的背景图 片,优点是非常便于控制位置，精灵图 也是一种运用场景）

        background-image: none/url(url[使用绝对或者相对地址指定背景图像]）

           <3>背景平铺background-repeat（默认情况下，背景图片都是平铺的）

        background-repeat: repeat/no-repeat/repeat-x/repeat-y;

        页面元素既可添加背景颜色，也可添加背景图片，只不过背景图片会压着背景颜色*********

          <4>背景图片的位置background-position

                background-position: x y;(x,y可以使用方位名词或者精确单位)

        （1）参数是方位名词

                *则前后的顺序无关，left top和top left的效果一样。

                *若只指定了一个方位名词，另一个省略，则第二个默认的是居中对齐。

        （2）参数是精确单位（第一个为X坐标，第二个为Y坐标，若有一个未写，默认为center）

        （3）参数是混合单位（第一个X，第二为Y）

<5>背景图像的固定（背景附着，图像是固定或者随着页面的其余部分滚动，常用于后期制作视差滚动的效果）

        background-attachment： scroll（默认：随着内容的滚动而滚动）/fixed

*****背景属性的复合写法：简写在一个background中，节约代码。

        习惯约定顺序：颜色，地址，平铺，滚动，位置

        例如：background: black url(文件位置) no-repeat fixed center top;
